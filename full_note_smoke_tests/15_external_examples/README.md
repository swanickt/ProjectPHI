# ProjectPHI External-Note Smoke Tests

## Scope

This smoke test uses a batch of 15 copied external de-identified note bodies,
with synthetic local row, patient, encounter, and note identifiers.

The selected examples are from the NCBI Open-Patients / TREC CDS 2016 subset listed
in `source_manifest.csv`. The Hugging Face dataset card listed in that manifest
describes the dataset as public patient notes.

## Files

- `source_manifest.csv`: source, URL, license, and source-note metadata for
  each selected external note body.
- `input_notes.csv`: 15 copied external de-identified note bodies with
  synthetic local metadata.
- `deidentified_output.csv`: pipeline output generated by the CLI, if the
  command below has been run.
- `audit_output.csv`: internal audit output generated by the CLI, if the
  command below has been run.
- `readable_review/`: optional human-readable copies of input notes,
  de-identified output notes, and audit rows if generated during manual review.

## Pipeline Command

```sh
time PROJECT_PHI_DATE_SHIFT_SECRET=external-local-date-secret \
uv run project-phi-deid \
  full_note_smoke_tests/15_external_examples/input_notes.csv \
  full_note_smoke_tests/15_external_examples/deidentified_output.csv \
  --audit-output-file full_note_smoke_tests/15_external_examples/audit_output.csv \
  --stable-date-shift \
  --date-shift-secret-env-var PROJECT_PHI_DATE_SHIFT_SECRET
```

The secret above is a synthetic local-evaluation value only.
